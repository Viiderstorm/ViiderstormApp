Checkpoint 2
Raytracing Framework - Jason Curry

Overview

This checkpoint marks the beginnings of the Ray Tracer. It sets the ground work for a basic Ray Tracer so that we can build on top of it and implement new features without having to refactor our code too much. It covers:

  • Creating an Image Plane
  • Creating Rays for each Pixel in the Image Plane relative to a camera
  • Computing ray-intersections for various geometries
  • Compiling the rays' colors into an image

The image we're rendering is our initial scene from checkpoint one. The colors are expected to appear flat since the ray tracer is using the object's material color as the computed color for the ray intersection. Soon we'll add highlights, reflections, and shadows to generate a more realistic image.
