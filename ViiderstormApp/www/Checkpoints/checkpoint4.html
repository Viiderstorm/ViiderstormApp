Checkpoint 4
Procedural Shading - Jason Curry

Overview

In this section, we're going to utilze a more advanced shading technique that allows us to procedurally create textures.

This checkpoint is very short and sweet. I've only made a few modifications from the previous checkpoint to be able to compute the colors on an intersection by intersection basis.

All I had to do was:

  • Create some new Material objects for the Geometries to use instead of Three.js's supplied materials
  • Give these new Material objects a function that will return the correct color given a position and an object
  • Add those objects to the Scene Geometry classes
  • Change the references to Three.js's material color so that instead reference the new getColor method

And it's done.

Always refactor your code when you have a chance. It really pays off if you constantly have to maintain, update, and add features to the code base.
