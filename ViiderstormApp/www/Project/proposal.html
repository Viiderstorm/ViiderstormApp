Project Proposal: Real-Time Path Tracing
by Jason Curry
CSCI-711 Global Illumination

Image of Bidirectional Path Tracing with 40 samples per pixel

Description

Path tracing is a computer graphics algorithm that is designed to realistically simulate light bouncing and reflecting within a scene. There are many algorithms that exist within the global illumination space that obtain equally or more realistic images, many of those extremely realistic algorithms being too computationally intensive to work in real-time for dynamic scenes.

This project aims to learn more about the potential of real-time global illumination by implementing a real-time path tracer. The project will be implemented using web technologies and libraries such as WebGL through Three.js. However, it is possible that there may be unforeseen roadblocks with the initially chosen libraries, so there's potential to change the utilized libraries later in development. The scene is expected to be rendered in real time with moveable geometries such that the scene is dynamic and interactive. The initial scene will be a sphere with editable materials in a multi-colored room with a single light source. The intent is to display the power of path tracing by showcasing realistic casting of shadows and reflections in real-time as well as the draw backs of this global illumination technique.

Project Obectives

  • Create a User Interface that allows real-time interaction with the scene
  • Implement Bidirectional Path Tracing
  • Examine the various methods of noise reduction if time permits
